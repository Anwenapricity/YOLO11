# 🎥 VOD — 视频目标检测输出目录

> **VOD** = **V**ideo **O**bject **D**etection 🎯  
> 本目录用于存放 **YOLO11 视频目标检测** 的推理结果，包括可视化视频、逐帧标注标签与逐帧检测图像。

---

## 📁 目录结构

```
VOD/
├── 🎬 {视频名}.avi          # 带检测框的可视化结果视频
├── 📂 images/               # 逐帧检测图像（含边界框）
│   ├── {视频名}_1.jpg       # 第 1 帧
│   ├── {视频名}_2.jpg       # 第 2 帧
│   └── ...
├── 📂 labels/               # 逐帧目标检测标签
│   ├── {视频名}_1.txt       # 第 1 帧
│   ├── {视频名}_2.txt       # 第 2 帧
│   └── ...
└── 📄 README.md             # 本说明文档
```

---

## 🎯 内容说明

| 类型 | 说明 |
|------|------|
| 🎬 **可视化视频** | 在原视频上叠加检测框与类别标签，便于直观查看检测效果 |
| 🖼️ **逐帧 Images** | 每一帧对应的检测图像，保存带框渲染结果，方便逐帧浏览与截图 |
| 🏷️ **逐帧 Labels** | 每一帧对应一个 `.txt` 文件，记录该帧中所有目标的边界框信息 |

---

## 📝 Labels 格式

每行代表一个检测到的目标，采用 **YOLO 检测格式**：

```
class_id x_center y_center width height
```

- 🏷️ `class_id` — 类别编号（COCO 80 类）
- 📐 `x_center y_center` — 边界框中心点坐标，**归一化** 至 `[0, 1]`（相对图像宽高）
- 📏 `width height` — 边界框宽高，**归一化** 至 `[0, 1]`

**示例：**

```
56 0.273336 0.264087 0.544920 0.528074
15 0.491359 0.658003 0.980911 0.675129
```

---

## 🚀 如何生成

运行项目根目录下的 `video_od.py` 即可将结果输出至本目录：

```bash
python video_od.py
```

默认使用 `yolo11m.pt` 模型，对输入视频逐帧推理并保存可视化视频、逐帧图像与 labels。

---

## 💡 小贴士

- 📌 标签与图像文件名 `{视频名}_{帧号}` 一一对应，便于帧级对齐
- 🔍 可用于后续分析、评估或转换为其他标注格式
- ⚙️ 推理参数（置信度阈值、模型权重等）可在 `video_od.py` 中调整
- 🎞️ 可视化视频格式为 `.avi`，可用常见播放器直接打开预览
