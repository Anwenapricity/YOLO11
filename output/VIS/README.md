# 🎬 VIS — 视频实例分割输出目录

> **VIS** = **V**ideo **I**nstance **S**egmentation 🎨  
> 本目录用于存放 **YOLO11 视频实例分割** 的推理结果，包括可视化视频与逐帧标注信息。

---

## 📁 目录结构

```
VIS/
├── 🎥 {视频名}.avi          # 带分割掩码的可视化结果视频
├── 📂 labels/               # 逐帧实例分割标签
│   ├── {视频名}_1.txt       # 第 1 帧
│   ├── {视频名}_2.txt       # 第 2 帧
│   └── ...
└── 📄 README.md             # 本说明文档
```

---

## 🎯 内容说明

| 类型 | 说明 |
|------|------|
| 🎥 **可视化视频** | 在原视频上叠加实例分割掩码与边界框，便于直观查看检测效果 |
| 🏷️ **逐帧 Labels** | 每一帧对应一个 `.txt` 文件，记录该帧中所有实例的分割多边形坐标 |

---

## 📝 Labels 格式

每行代表一个检测到的实例，采用 **YOLO 分割格式**：

```
class_id x1 y1 x2 y2 x3 y3 ...
```

- 🏷️ `class_id` — 类别编号（COCO 80 类）
- 📐 `x1 y1 x2 y2 ...` — 分割多边形顶点坐标，**归一化** 至 `[0, 1]`（相对图像宽高）

**示例：**

```
56 0.136111 0.357031 0.133333 0.358594 ...
15 0.330556 0.365625 0.329167 0.366406 ...
```

---

## 🚀 如何生成

运行项目根目录下的 `video_seg.py` 即可将结果输出至本目录：

```bash
python video_seg.py
```

默认使用 `yolo11m-seg.pt` 模型，对输入视频逐帧推理并保存可视化视频与 labels。

---

## 💡 小贴士

- 📌 标签文件名 `{视频名}_{帧号}.txt` 与视频帧一一对应
- 🔍 可用于后续分析、评估或转换为其他标注格式
- ⚙️ 推理参数（置信度、设备等）可在 `video_seg.py` 中调整
